1
Create a Google Sheet
~1 min
→
Open Google Sheets

Go to sheets.google.com and sign in with your Google account.

→
Create a new spreadsheet

Click the large + (Blank) button to create a new spreadsheet. Rename it Masraf by clicking "Untitled spreadsheet" at the top left.

→
Leave it open

Keep this tab open — you'll need it in the next step.

2
Add the backend script
~2 min
→
Open Apps Script

In your Google Sheet, click the menu: Extensions → Apps Script. A new tab will open with a code editor.

→
Replace the existing code

Select all the existing code in the editor (Ctrl+A or Cmd+A), then delete it. Copy the code below and paste it in.

Code.gs — Apps Script 
// Paste the full contents of Code.gs here.
// Get the latest version from the repository.
→
Save the script

Click the floppy disk icon or press Ctrl+S / Cmd+S.

→
Run it once to grant permissions

Click the ▶ Run button. Google will ask for permissions — click Review permissions, choose your account, then click Advanced → Go to Masraf (unsafe) → Allow.

This warning is normal — it just means the script was not published by Google. It is your own code running in your own account.
3
Deploy as a web app
~1 min
→
Open the deploy dialog

In the Apps Script editor, click Deploy (top right) → New deployment.

→
Select Web app as the type

Click the gear ⚙ icon next to "Select type" and choose Web app.

→
Configure the settings

Fill in as follows:

Description: Masraf

Execute as: Me

Who has access: Anyone

Why "Anyone"? The app runs from your phone without being signed in to Google. Setting access to "Anyone" allows it to communicate with your sheet silently. The URL is a long random string — nobody can guess it.
→
Copy the Web App URL

Click Deploy. Google will show you a URL that looks like:

https://script.google.com/macros/s/AKfyc.../exec ←
Save this URL. You will need it in the next step. If you lose it, see Section 10 to retrieve it.
4
Connect the app to your sheet
~30 sec
→
Open the app

Go to the Masraf app URL in your browser. You will see the connection screen.

→
Paste the Web App URL

Paste the URL you copied in Step 3 into the input field on the setup screen.

→
Tap Connect & Start

The app will connect to your Google Sheet. The sync dot in the top-right corner will turn green once the connection is confirmed. You are ready to go.

Using on multiple devices? Just open the app URL on any other phone or computer and paste the same Web App URL. All your data will load automatically.
Using the app
5
Add to your home screen

For the best experience, add Masraf to your phone home screen so it opens fullscreen like a native app.

iPhone (Safari)
  • Open the app URL in Safari
  • Tap the Share button
  • Tap "Add to Home Screen"
  • Tap Add
Android (Chrome)
  • Open the app URL in Chrome
  • Tap the 3-dot menu
  • Tap "Add to Home Screen"
  • Tap Install
6
Logging an expense
1
Type the category

Enter a short label — e.g. Benzine, Groceries, Gym.

2
Enter the amount

Type the amount and choose the currency using the LL / $ toggle on the right. A live conversion preview appears below the field.

3
Set the date

Use the three dropdowns to pick the day, month, and year. They default to today so you usually don't need to change anything.

4
Tap + Add

The entry saves instantly to Google Sheets and appears grouped under its day, showing both the L.L. and $ amounts.

7
Currency & exchange rate
→
Updating the rate

The exchange rate is shown in the top-right corner as LL/$. Tap the number to edit it. The new rate is saved to your sheet and applies to all future entries.

→
Entering in dollars

Switch the toggle to $ before typing. Masraf converts the amount to L.L. using the current rate before saving. All totals are always shown in both currencies.

8
Browsing & filtering
→
Filter by month

On phone: use the Month dropdown above the entries list. On desktop: click any month in the left sidebar. Select All time to see all entries across every month.

→
Deleting an entry

Tap the × on the right of any entry to remove it. The deletion syncs to your sheet immediately.

The dot in the top-right corner shows sync status:

Green — saved and up to date
Blue (pulsing) — currently saving
Red — connection error, check internet
9
Summary & export
→
Summary tab

Switch to the Summary tab (or panel on desktop) to see a formatted text summary of the selected month — matching the Google Keep note style.

→
Copy to clipboard

Tap Copy to clipboard to copy the full summary and paste it into Google Keep, WhatsApp, or anywhere else.

10
Retrieve your connection URL

If you need the Apps Script URL again (new phone, new browser, or you cleared the app), here is how to find it without creating a new deployment.

1.Open your Masraf Google Sheet
2.Click Extensions → Apps Script
3.Click Deploy (top right) → Manage deployments
4.Your existing deployment is listed — click the copy icon next to the Web App URL
5.Paste that URL into the app setup screen and tap Connect & Start
Important: always use Manage deployments to retrieve your URL. Do not click "New deployment" — that generates a different URL and disconnects any devices using the old one.
